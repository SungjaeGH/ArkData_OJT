## 1. Anagram Checker
### ✔ Anagram?
- 한 단어의 철자를 분해해 다른 단어, 혹은 다른 문장으로 바꾸는 놀이

### ✔ 과제
- Anagram의 정합성을 체크하기 위한 프로그램을 작성
- Output은 Input이 올바른지 올바르지 않은 지 검증 결과를 알려준다.
- O(N)으로 풀 것

### ✔ 조건
- 문자의 제한은 150글자로 제한
- 2개의 문자를 입력 받음
- 유니코드, 특수문자는 검증에서 제외 (알파벳과 숫자만 허용)
- 알파벳 대소문자는 구분하지 않는다. ('A' == 'a')
- 두 개의의 값을 서로 비교하지 않고 처리할 것
